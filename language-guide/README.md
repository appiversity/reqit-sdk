# The Reqit Language Guide

**A comprehensive guide to writing academic requirements in Reqit**

---

## Who This Guide Is For

This guide is for registrars, academic advisors, department chairs, and anyone responsible for defining and maintaining academic requirements. You don't need a programming background — Reqit is designed to read like structured English. If you can write catalog copy, you can write Reqit.

## How To Use This Guide

Start with the [Introduction](01-introduction.md) to understand what Reqit is and why it matters. Then work through the chapters in order — each one builds on the previous. By the end, you'll be able to write requirements for any academic program, from a simple course prerequisite to a full degree with general education, concentrations, and graduation requirements.

Every chapter includes real-world examples based on actual university requirements. You'll see familiar patterns from your own institution.
